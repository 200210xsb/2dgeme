## 2. Boss 血条 UI ❤️

### 功能特性
- **位置**: 屏幕顶部中央
- **显示时机**: Boss 受到伤害时自动显示
- **消失时机**: Boss 死亡后自动隐藏
- **低血量警告**:
  - HP < 30%: 红色文字
  - HP < 60%: 橙色文字
  - HP > 60%: 白色文字

### UI 组成
```
UI/
├── BossHpBarBg/         # 血条背景（深灰色）
│   └── BossHpBarFill/   # 血量填充（红色）
└── BossHpText/          # 血量文字 "BOSS HP: 8/10"
```

### 脚本逻辑
```gdscript
# BossHealthBar.gd
- 监听 Boss 的 damaged 信号
- 实时更新血条宽度
- 根据血量比例改变颜色
- Boss 死亡后自动隐藏
```

### 显示规则

- **时间**: 每个图标下方显示剩余秒数

- **自动清理**: 效果结束后移除对应图标

`StatusEffectsUI.gd` - 每帧检测玩家状态效果并更新 UI

## 4. 武器专属特效 ⚔️

### 6 种武器特效对比

#### Sword (剑) 🗡️
- **颜色**: 金黄色 `RGB(1.0, 0.8, 0.2)`
- **尺寸**: 中等 (3px)
- **时长**: 0.15s
- **形状**: 弧形斩击轨迹
- **特点**: 平衡型，经典剑光

#### Spear (矛) 🔱
- **颜色**: 冰蓝色 `RGB(0.6, 0.8, 1.0)`
- **尺寸**: 大 (4px)
- **时长**: 0.18s
- **形状**: 直线突刺 + 矛尖光点
- **特点**: 长距离直线攻击

#### Hammer (锤) 🔨
- **颜色**: 土黄色 `RGB(0.8, 0.6, 0.4)`
- **尺寸**: 最大 (5px)
- **时长**: 0.20s
- **形状**: 圆形冲击波
- **特点**: 重击地面效果

#### Dagger (匕首) 🗡️
- **颜色**: 银白色 `RGB(0.9, 0.9, 0.9)`
- **尺寸**: 最小 (2px)
- **时长**: 0.10s
- **形状**: 快速 X 形短斩
- **特点**: 快速连续攻击

#### Axe (斧头) 🪓
- **颜色**: 橙红色 `RGB(1.0, 0.4, 0.2)`
- **尺寸**: 大 (5px)
- **时长**: 0.22s
- **形状**: 重斧劈砍弧线 + 冲击点
- **特点**: 缓慢但强力

#### Bow (弓) 🏹
- **颜色**: 浅绿色 `RGB(0.8, 1.0, 0.6)`
- **尺寸**: 小 (2px)
- **时长**: 0.12s
- **形状**: 射击轨迹线 + 箭矢光晕
- **特点**: 远程射击特效

### 实现方式
```gdscript
# AttackEffect.gd
var weapon_configs = {
    "sword": {...},
    "spear": {...},
    // ... 6 种武器配置
}

func play(position, dir, w_type):
    var config = weapon_configs[w_type]
    effect_color = config["color"]
    pixel_size = config["size"]
    duration = config["duration"]
    # 根据类型绘制不同形状
```
