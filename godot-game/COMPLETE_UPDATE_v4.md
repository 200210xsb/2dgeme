## 1. 陷阱系统激活 🗺️

### 陷阱类型
| 类型 | 外观 | 伤害 | 出现章节 |
|------|------|------|----------|
| **尖刺** | 灰色三角形尖刺 | 随关卡增加 | Chapter 1+ |
| **落石** | 棕色圆形岩石 | 随关卡增加 | Chapter 3+ |
| **岩浆** | 橙红色波浪表面 | 随关卡增加 | Chapter 5+ |

### 生成机制
- 每 **8 秒** 随机生成一个陷阱
- 陷阱在战场中随机位置出现（x: 200-1000）
- 20 秒后自动消失

### 实现代码
```gdscript
# Game.gd
func _update_trap_spawning(delta: float):
    trap_spawn_timer += delta
    if trap_spawn_timer >= 8.0:
        _spawn_random_trap()

func _spawn_random_trap():
    var trap_type = [0, 1, 2].pick_random()  # 尖刺/落石/岩浆
    var trap = TrapSystem.new()
    trap.position = Vector2(rand_range(200, 1000), 580)
    add_child(trap)
```

## 3. 完美格挡/弹反系统 🛡️

### 格挡机制

#### 普通格挡
- **触发**：按住 Block 键
- **方向要求**：面向敌人 ±15°内
- **减伤效果**：75%（只受 25% 伤害）
- **击退减少**：70%

#### 完美格挡（弹反）
- **触发时机**：受击前 **0.15 秒** 内按下格挡
- **效果**：
  - 完全免疫伤害（0 伤害）
  - 0 击退
  - 0.2 秒无敌时间
  - 蓝色冲击波特效
  - 120ms 命中停顿
  - 播放攻击音效（弹反成功提示）

### 实现代码
```gdscript
# Player.gd
var perfect_block_timer := 0.0
var perfect_block_window := 0.15

# 格挡输入时启动完美格挡窗口
if block_pressed:
    perfect_block_timer = perfect_block_window

# 受击时判定
if perfect_block_timer > 0 and angle_diff < block_angle:
    # 完美格挡成功
    amount = 0
    knockback = 0
    hit_effects.spawn_shockwave(..., Color(0.2, 0.8, 1.0, 0.8))
    hit_stop_manager.trigger_hit_stop(0.12)
```

### 高级技巧
- **连续格挡**：持续按住可保持普通格挡
- **弹反窗口**：看到攻击前摇时按下
- **弹反后反击**：弹反成功后立即攻击

#### 4.1 远程弹幕 💜
- **冷却**：4.0 秒
- **伤害**：1 点
- **速度**：320 像素/秒
- **触发条件**：
  - Boss 进入 Phase 2（红血）
  - 距离玩家 1.5-2.0 倍攻击范围
  - 不在近战范围内

**视觉效果**：
- 紫色魔法球（外圈 + 内核双层）
- 射击时产生火花特效
- 直线飞行轨迹

#### 4.2 蓄力冲撞 🔴
- **冷却**：6.0 秒
- **伤害**：3 点
- **速度**：520 像素/秒
- **触发条件**：
  - Boss 进入 Phase 2
  - 距离玩家 > 2 倍攻击范围

**技能流程**：
1. **蓄力阶段**（0.6 秒）：Boss 停顿，红色冲击波预警
2. **冲刺阶段**：高速直线冲撞
3. **命中/结束**：造成伤害或冲出屏幕

**应对策略**：
- 看到红色预警时向反方向移动
- 可以跳跃躲避
- 完美格挡可完全防御

#### 4.3 自我治疗 💚
- **冷却**：15.0 秒
- **治疗量**：3 HP
- **触发条件**：Phase 2，低血量时

### Boss 技能时间轴（Phase 2）

## 5. 状态效果系统 ⚡

### 效果类型

| 效果 | 图标 | 伤害/秒 | 持续时间 | 特殊效果 |
|------|------|---------|----------|----------|
| **中毒** ☠️ | 绿色 | 1 | 3 秒 | - |
| **燃烧** 🔥 | 橙色 | 2 | 2 秒 | - |
| **冰冻** ❄️ | 蓝色 | 0 | 1.5 秒 | 减速 50% |
| **眩晕** 💫 | 黄色 | 0 | 1.0 秒 | 无法行动 |
| **流血** 🩸 | 红色 | 1 | 4 秒 | - |

### 触发机制
- **触发概率**：20%（暴击或连击时）
- **持续时间**：2 秒（可叠加）
- **叠加规则**：刷新时间 + 叠加层数

### 应用对象
- ✅ 玩家（受到敌人攻击时）
- ✅ 敌人（受到玩家攻击时）
- ✅ Boss（受到玩家攻击时）

### 实现代码
```gdscript
# Game.gd - 攻击命中时
if randf() < 0.2 and body.has_node("StatusEffects"):
    var effect_types = [1, 2, 4]  # BURN, FREEZE, BLEED
    var effect = effect_types.pick_random()
    body.status_effects.apply_effect(effect, 2.0)

# Player.gd / Enemy.gd - 每帧处理
if status_effects != null:
    var effect_damage = status_effects.process_effects(delta)
    if effect_damage > 0:
        hp -= effect_damage
```

### 战术应用
- **燃烧**：快速消耗 Boss 血量
- **冰冻**：减速 Boss，争取输出时间
- **流血**：持续伤害，适合游击战术
- **中毒**：均衡 DOT 伤害
- **眩晕**：打断 Boss 技能（未来扩展）
