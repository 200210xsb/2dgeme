## 🎮 游戏技巧

### 新手入门
1. 先用 **Sword** 熟悉操作
2. 学会 **格挡** 减少伤害
3. 连续攻击积累 **连击数**
4. 通关后到 **商店升级**

### 进阶技巧
1. **完美格挡**: 观察 Boss 攻击前摇，提前 0.15s 按 K
2. **连击维护**: 2 秒内命中敌人，否则连击中断
3. **武器克制**: 
   - 近战小怪用 Dagger 快速清理
   - Boss 用 Axe 高伤害或 Bow 远程
4. **状态效果**: 连击时有 20% 概率触发，配合高攻速武器
5. **陷阱利用**: 引敌人踩尖刺/落石

### 高手向
1. **弹反流**: 完美格挡后反击，Boss 硬直
2. **状态流**: Dagger + 状态触发，DOT 伤害
3. **连击流**: 快速武器叠连击，+100% 伤害
4. **坦克流**: 升级 HP + 格挡，硬刚 Boss
