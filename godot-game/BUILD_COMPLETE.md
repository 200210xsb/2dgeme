## 游戏操作

### 默认键位
| 按键 | 功能 |
|------|------|
| A/D | 左右移动 |
| W | 跳跃 |
| J | 攻击 |
| Q | 切换武器 |
| R | 重开 |
| N | 下一关（通关后） |
| 1-5 | 商店购买 |

### 开场选择
| 按键 | 功能 |
|------|------|
| 方向键左/右 | 切换章节 |
| 方向键上/下 | 切换关卡 |
| Enter | 开始游戏 |

### 商店购买
| 按键 | 功能 |
|------|------|
| 1 | HP 升级 (40+20*lv) |
| 2 | 攻击升级 (50+25*lv) |
| 3 | 满血回复 (25 金币) |
| 4 | 攻速升级 (60+28*lv) |
| 5 | 暴击升级 (70+34*lv) |
| N | 进入下一关 |

## 游戏配置

### 修改键位
编辑 `keybinds.cfg` 文件：
```ini
[keys]
ui_left="Left"
ui_right="Right"
ui_up="Up"
ui_down="Down"
attack="J"
switch_weapon="Q"
restart="R"
shop_buy_hp="1"
shop_buy_attack="2"
shop_buy_heal="3"
shop_buy_speed="4"
shop_buy_crit="5"
```

### 故事背景
游戏设定在"裂界战争"后的世界，十位领主割据十域。玩家扮演守誓者，需要穿越十章战线，逐步修复核心并终结裂隙。

## 游戏机制

### 武器系统
| 武器 | 攻速 | 连击伤害 | 特点 |
|------|------|----------|------|
| 剑 | 快 | 1/2/3 | 平衡 |
| 矛 | 中 | 1/2/2 | 范围较远 |
| 锤 | 慢 | 2/3/4 | 单发高伤 |

### 暴击系统
- 基础暴击率：5% * 暴击等级
- 暴击伤害倍率：160% + 5% * 暴击等级
- 最大暴击率：45%

### Boss 机制
- **P1 阶段**：常规攻击、冲刺
- **P2 阶段**（HP < 50%）：
  - 移速提升 35%
  - 伤害增加
  - 解锁范围 AOE 技能
  - 解锁召唤小怪

### 升级效果
- **HP**: +1 最大 HP 每级
- **攻击**: +12% 伤害 每级
- **攻速**: -6% 攻击间隔 每级（最大 -45%）
- **暴击**: +5% 暴击率 每级（最大 45%）
