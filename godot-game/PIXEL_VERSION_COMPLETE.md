## 🖼️ 像素风格特性

### 像素角色
- **玩家**: 9x14 像素蓝色小人
- **敌人**: 8x12 像素红色方块
- **Boss**: 12x18 像素大魔王
- **眼睛**: 白色/橙色像素点

### 像素特效
- **攻击**: 金色 8x10 像素弧线
- **伤害**: 5x7 像素数字字体
- **死亡**: 16 个方形像素粒子
- **受击**: 整体红色闪烁

### 像素背景
- **天空**: 蓝色渐变像素带
- **山脉**: 24px 方块层叠
- **地面**: 16px 像素纹理
- **视差**: 山脉和地面对比滚动

### 渲染设置
- 纹理过滤：**关闭**
- 拉伸模式：**viewport**
- 保持像素**锐利无模糊**

## 🎯 技术实现

### 程序化像素画
使用字符串定义像素图案：
```gdscript
var sprite_pixels = [
    "000011000",
    "000011000",
    "000111100",
    "001111110",
    ...
]
```

### 像素渲染
- 每个 "1" 绘制一个彩色矩形
- 通过 `pixel_size` 控制放大倍数
- 关闭纹理过滤保持锐利

### 动态效果
- 颜色变化（受击、狂暴）
- 方向翻转
- 透明度渐变
- 粒子物理

## 🎨 像素艺术原则

### 设计哲学
1. **简约** - 用最少的像素表达特征
2. **对比** - 高对比突出轮廓
3. **识别** - 关键特征放大
4. **清晰** - 动作反馈明显
